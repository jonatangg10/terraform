Instalacion Windows

Nota: En los portatiles de seti no deja.

https://developer.hashicorp.com/terraform/install#windows

386 : Windows 32 bits
AMD : Windows 64 bits

Descargar.
Al tener el zip, creamos un carpeta de terraform , movemos el zip a la carpeta creada.
Descomprimimos el zip, al tener el archivo .exe lo abrimos.
Una vez abierto el el archivo .exe, habrimos un cmd, entramos por medio de comandos a la carpeta 
creada, al estar en la ruta correcta.

Escribimos: terraform.exe --version
Al haber dado enter, nos debe dar la version.

Si salimos de la carpeta creada, y ejecutamos el mismo codigo terraform.exe --version.
No lo va a reconocer porque solo lo reconoce en la carpeta en la que lo ejecutamos.
Para que lo reconosca en otras carpetas.
Copiamos el archivo ejecutable .exe.
Entramos a la carpeta Windows/System32 y lo pegamos.

Ahora desde cualquier lugar se puede ejecutar.
terraform --version

Instalacion Linux

https://developer.hashicorp.com/terraform/install#linux

Seleccionamos Ubuntu/Debian
AMD64 , Click derecho , Copiar dirección del vinculo.

Abrimos la Konsole en nuestro Ubuntu

Entramos en el usuario root
sudo su 
cd /root/

Ejecutamos el siguiente comando para la instalación de terraform.
wget https://releases.hashicorp.com/terraform/1.7.3/terraform_1.7.3_linux_amd64.zip

ls
Miramos si el archizo .zip esta descargado

Ahora hacemos unzip
unzip terraform_1.7.3_linux_amd64.zip

ls
Revisamos que halla un archivo nuevo.
El archivo nuevo se llama: terraform

Ahora ejecuramos el el archivo
./terraform --version
Nos muestra la version del terraform instalado.

Si cambiamos de ruta 
cd /ect/
escribimos terraform
No lo va a reconocer, por que no esta ejecutado globalmente en el sistema.

Para que se reconozca globalmente.
Volvemos a donde descomprimimos el archivo.
Ejecutamos el siguiente codigo.
cp terraform /usr/bin/

Una vez hecho el paso anterior, probamos desde otra ruta la version de terraform instalada.
cd /home/ 
terraform --version
Nos muestra la version del terraform instalado.
